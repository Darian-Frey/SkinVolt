# ⚡ SkinVolt
### *High‑voltage Steam Market intelligence.*

SkinVolt is a fast, cross‑platform Steam Market analytics app built with **Rust + Tauri**. It delivers clean financial‑grade insights for traders, collectors, and market analysts — including live pricing, volatility metrics, portfolio tracking, and advanced item analytics.

---

## 🔍 Features
- Live Steam Market price tracking  
- Historical charts with moving averages & volatility  
- Portfolio valuation and inventory import  
- Price alerts with background refresh  
- Expected value tools (cases, trade‑ups, collections)  
- Sentiment indicators & manipulation detection  
- Clean, premium analytics‑focused UI  

---

## 🧱 Tech Stack
- **Rust** — backend engine, analytics, caching  
- **Tauri** — lightweight cross‑platform desktop shell  
- **HTML/CSS/JS** — UI layer  
- **SQLite / sled** — local data storage  

---

## 🚧 Status
SkinVolt is currently in early development. Core systems, analytics modules, and UI scaffolding are being built according to the project roadmap.

---

## 📦 Goals
- Deliver a professional‑grade Steam Market tracker  
- Provide advanced analytics not found in typical market apps  
- Offer a clean, financial‑style interface with optional themes  
- Support Linux and Windows with native performance
